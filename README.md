## Features

- [Tailwind CSS](https://tailwindcss.com)
- Almost typical to Spotify
- Material Icons
- Desktop responsive


## Getting started

**I use yarn** 

- For development

  `yarn build` Or `yarn build --watch`

- For Production
    
    `yarn production`
